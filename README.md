# ZeBusRoute

Mobilna aplikacija za praćenje javnog prevoza u gradu Zenici, razvijena koristeći .NET MAUI i C#.

## O projektu

ZeBusRoute je mobilna aplikacija koja omogućava građanima Zenice da jednostavno prate autobuse javnog prevoza, pregledaju linije i kupuju karte direktno kroz aplikaciju. Cilj projekta je digitalizacija i poboljšanje korisničkog iskustva javnog prevoza u gradu.

## Glavne funkcionalnosti

- **Pretraga i prikaz linija**: Pregled svih dostupnih bus linija sa detaljnim rasporedima
- **Rasporedi vožnje**: Pristup ažuriranim voznim redovima za sve linije
- **Kupovina karata**: Mogućnost kupovine karti direktno kroz aplikaciju

## Tehnologije

- **.NET MAUI** - Cross-platform framework za razvoj mobilnih aplikacija
- **C#** - Programski jezik
- **GPS/Geolokacija** - Za praćenje lokacije autobusa

## Preduslov za pokretanje

- Visual Studio 2022 ili noviji
- .NET 8.0 SDK ili noviji
- Android SDK (za Android development)
